## Tree to Linked List (Flatten Binary Tree)

### Concept:
Convert a binary tree into a linked list using preorder traversal.

### Key Idea:
- Use recursion
- Maintain previous pointer
- Rearrange left and right pointers

### Time Complexity:
O(n)

### Space Complexity:

- Flatten Binary Tree
